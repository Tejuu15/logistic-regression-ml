# Logistic Regression - Titanic Dataset

## Objective

To build a binary classification model using logistic regression.

## Tools Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

## Dataset

Titanic Dataset

## Steps Performed

1. Data preprocessing and cleaning
2. Feature selection and scaling
3. Train-test split
4. Model training using Logistic Regression
5. Evaluation using confusion matrix, precision, recall, ROC-AUC

## Results

* Model successfully classified survival
* Accuracy, precision, and recall calculated
* Confusion matrix used for evaluation

## Conclusion

Logistic regression is effective for binary classification problems and provides probability-based predictions.
